Perceivable

Information and user interface components must be presentable to users in ways they can perceive.

Guideline 1.1: Text Alternatives

Provide text alternatives for any non-text content so that it can be changed into other forms people need, such as large print, braille, speech, symbols or simpler language.

non-text content

non-text contents is explained by w3c as: any content that is not a sequence of characters that can be programmatically determined orwhere the sequence is not expressing something in human language. This includes ASCII Art (which is a pattern of characters), emoticons, leetspeak (which uses character substitution), and images representing text

Examples:

1. Missing alt attribute

Correct

This image has an alt attribute that explains the content of the image.

Wrong

This image does not have an alt attribute.

2. Decorative image containing alt with non-related text.

Correct

This image is decorative and should therefore contain an empty alt attribute.

Wrong

This image is decorative but contains an alt attribute that says "This tag is not necessary".

3. Image containing text that is important for the reader.

Correct

This image contains important text that should be conveyed in the alt attribute.

Wrong

This image contains important text but the text is not conveyed in the alt attribute.
